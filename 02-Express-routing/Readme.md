# Backend with Luv

A simple backend server built using Node.js, TypeScript, and Express.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/luvsharma6342/backend-series-luv.git
cd backend-series-luv
cd 02-Express-routing
```

### 2. Install Dependencies
npm install

### 3. Run the Server
npm run dev

## 🛠 Tech Stack
- Node.js
- TypeScript
- Express.js

## 📌 Notes
- Make sure Node.js is installed on your system
- This project is part of a backend development series
